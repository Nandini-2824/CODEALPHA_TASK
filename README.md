# 🎮 Hangman Game (Console Version)

A simple **text-based Hangman game** built using Python.
The player attempts to guess a randomly selected word, one letter at a time, with a limited number of incorrect guesses.

---

## 🎯 Project Goal

Create a basic Hangman game that runs in the **console**, focusing on core programming concepts like loops, conditionals, and data structures.

---

## ⚙️ Features

* 🔤 Random word selection from a predefined list of 5 words
* ⌨️ Letter-by-letter user input via console
* ❌ Maximum of 6 incorrect guesses allowed
* 📊 Displays progress after each guess
* 🏁 Win/Lose condition handling
* 🔁 Continuous gameplay using loops

---

## 🧠 Key Concepts Used

* `random` module (for word selection)
* `while` loop (game loop control)
* `if-else` statements (decision making)
* Strings (word handling)
* Lists (storing words and guesses)

---

## 🎮 How to Play

* The program selects a random word from a list
* You guess one letter at a time
* Correct guesses reveal letters in the word
* Incorrect guesses reduce your remaining attempts
* You lose after **6 wrong guesses**
* You win if you guess the word before running out of attempts
